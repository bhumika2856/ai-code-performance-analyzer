# AI Code Performance Analyzer

An AI-assisted static analysis tool that predicts the time complexity of C/C++ programs.

## Features

- Static code analysis for C/C++ programs
- Feature extraction from source code
- Machine learning based complexity prediction
- Automated performance insights

## Tech Stack

Python  
Scikit-learn  
Static Code Analysis  
C/C++ Parsing  

## Project Structure

dataset/ - labeled algorithm dataset  
parser/ - code parsing modules  
feature_extractor/ - feature engineering from code  
model/ - machine learning models  
evaluation/ - model testing  
reports/ - generated analysis reports  

## Goal

Predict algorithm complexity such as:

O(1)  
O(log n)  
O(n)  
O(n log n)  
O(n²)
